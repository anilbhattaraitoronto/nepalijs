Beginner Lessons: Intro

Nepali language is the first language of over 33 million people. As any other language, it's formal structure is built up from vowels, root syllables, syllables, words, phrases, sentences and paragraphs. In the following seven lessons, we will learn these components from the smallest units to gradually more complex components.

Nepali has

  • 12 Root Vowels, and
  • 36 Root Syllables

All of formal Nepali language is composed from these basic building blocks.

By the end of the seven lessons, you should be able to communicate in basic written Nepali (reading and writing). We will have separate components for practice-writing blogs for signed-up users.

That's enough of basic intro. We are here to learn.

Let's start!

Root Syllables

Root syllables are formed by joining the first vowel with 36 root consonants or half root syllables.

For example,

  • क् + अ = क (ka)
  • ख् + अ = ख (kha)
and, so on.

The next lesson has 36 root syllables. You can easily guess how the rest of the root syllables are formed.

Extended Root Syllables

In the previous lesson, we learned the 36 root syllables by combining the 36 roots with the first voewl, 'अ'.

In the following session, we will expand that to create extended root syllables by combining the roots with all the 12 vowels.

The following list provides a glimpse of how that looks like for the first root 'क'.

  • क् + अ = क (ka)
  • क् + आ = का (kaa)
  • क् + इ = कि (ki)
  • क् + ई = की (kee)
  • क् + उ = कु (ku)
  • क् + ऊ = कू (koo)
  • क् + ए = के (ke)
  • क् + ऐ = कै (kai)
  • क् + ओ = को (ko)
  • क् + औ = कौ (kau)
  • क् + अं = कं (kam)
  • क् + अ: = क: (kaha)

Now You can try making extended syllables for the rest of the 35 roots. The process is exactly the same as for क, except you will replace क with other roots such as ख, ग, घ, ङ, etc.

Practice Extended Root Syllables

Now is the time to get your hands dirty. Now that you know the logic behind the generation of extendes Root Syllables, please practice making extended root syllables for other roots.

Vowels and syllables (both root and extended root) are the building blocks of words and sentences. That's where we will move in the next lessons.

The next lessons will focus on practicing words.We will begin with very commone words. Following words, we will start learning sentences.

The order of composition in Nepali sentence is: Subject + Object + Verb. We will learn four different types of words: nouns/pronouns; verbs; adjectives; adverbs. If you know the order of sentence composition, you can try making up your own sentences even before you delve into sentence learning.

We will move from easy words to difficult ones.
