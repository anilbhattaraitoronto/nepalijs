Learn Nepali

Lesson 1: Basic Intro

Nepali language is the first language of over 33 million people. As any other language, it's formal structure is built up from vowels, root syllables, syllables, words, phrases, sentences and paragraphs. In the following seven lessons, we will learn these components from the smallest units to gradually more complex components.

Nepali has

  • 12 Root Vowels, and
  • 36 Root Syllables

All of formal Nepali language is composed from these basic building blocks.

By the end of the seven lessons, you should be able to communicate in basic written Nepali (reading and writing). We will have separate components for practice-writing blogs for signed-up users.

That's enough of basic intro. We are here to learn.

Let's start!
